Introducción

Bienvenido al manual de HTML. En este documento, aprenderás sobre las etiquetas más importantes de HTML y cómo usarlas correctamente.

Etiqueta <html>

La etiqueta <html> es el contenedor principal de un documento HTML. Todo el contenido de la página debe estar dentro de esta etiqueta.

Ejemplo:

<html>
  <head>
    <title>Título del documento</title>
  </head>
  <body>
    <p>Contenido de la página</p>
  </body>
</html>

Etiqueta <head>

La etiqueta <head> contiene meta información sobre el documento, como el título, enlaces a hojas de estilo y scripts.

Ejemplo:

<head>
  <title>Título del documento</title>
  <link rel="stylesheet" href="styles.css">
  <script src="script.js"></script>
</head>

Etiqueta <body>

La etiqueta <body> contiene el contenido principal del documento HTML, como texto, imágenes, enlaces, etc.

Ejemplo:

<body>
  <h1>Título Principal</h1>
  <p>Este es un párrafo de texto.</p>
  <img src="imagen.jpg" alt="Descripción de la imagen">
</body>

Etiqueta <div>

La etiqueta <div> se utiliza como contenedor genérico para agrupar contenido. No tiene un significado específico, pero es útil para aplicar estilos y scripts a bloques de contenido.

Ejemplo:

<div class="contenedor">
  <h2>Subtítulo</h2>
  <p>Este es un párrafo dentro de un div.</p>
</div>

Etiqueta <span>

La etiqueta <span> se utiliza para aplicar estilos a partes específicas del contenido sin crear una nueva línea. Es un contenedor en línea.

Ejemplo:

<p>Este es un <span style="color: red;">texto en rojo</span> dentro de un párrafo.</p>

Etiqueta <a>

La etiqueta <a> define un hipervínculo que enlaza a otra página o sección dentro del mismo documento.

Ejemplo:

<a href="https://www.ejemplo.com">Ir a Ejemplo.com</a>

Etiqueta <p>

La etiqueta <p> se utiliza para definir un párrafo de texto. Los navegadores añaden un margen antes y después de cada párrafo por defecto.

Ejemplo:

<p>Este es un párrafo de texto.</p>

Etiquetas de encabezado

Las etiquetas de encabezado <h1> a <h6> se utilizan para definir títulos y subtítulos en el contenido. <h1> es el más importante y <h6> es el menos importante.

Ejemplo:

<h1>Título principal</h1>
<h2>Subtítulo de primer nivel</h2>
<h3>Subtítulo de segundo nivel</h3>
<h4>Subtítulo de tercer nivel</h4>
<h5>Subtítulo de cuarto nivel</h5>
<h6>Subtítulo de quinto nivel</h6>
