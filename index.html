Etiqueta <html>

La etiqueta <html> es el contenedor principal para todo el contenido HTML. Debe contener las etiquetas <head> y <body>.

<html>
    <head>
        <title>Título de la página</title>
    </head>
    <body>
        <h1>Encabezado</h1>
        <p>Parrafo de ejemplo.</p>
    </body>
</html>

Etiqueta <head>

La etiqueta <head> contiene metadatos y enlaces a recursos externos como hojas de estilo y scripts.

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título de la página</title>
    <link rel="stylesheet" href="styles.css">
</head>

Etiqueta <body>

La etiqueta <body> contiene el contenido principal de la página web que se muestra en el navegador.

<body>
    <h1>Bienvenidos</h1>
    <p>Este es el contenido principal de la página.</p>
</body>

Etiqueta <div>

La etiqueta <div> se utiliza para crear contenedores genéricos en la página, para agrupar otros elementos y aplicar estilos o scripts.

<div>
    <h2>Sección de ejemplo</h2>
    <p>Este es un párrafo dentro de un div.</p>
</div>

Etiqueta <span>

La etiqueta <span> se utiliza para aplicar estilos a partes del texto sin crear un nuevo bloque. Es un contenedor en línea.

<p>Este es un <span style="color: red;">texto resaltado</span> en el párrafo.</p>

Etiqueta <h1> a <h6>

Las etiquetas <h1> a <h6> se utilizan para definir encabezados, siendo <h1> el más importante y <h6> el menos importante.

<h1>Encabezado 1</h1>
<h2>Encabezado 2</h2>
<h3>Encabezado 3</h3>
<h4>Encabezado 4</h4>
<h5>Encabezado 5</h5>
<h6>Encabezado 6</h6>

Etiqueta <p>

La etiqueta <p> se utiliza para definir párrafos de texto.

<p>Este es un párrafo de ejemplo.</p>

Etiqueta <a>

La etiqueta <a> define un hipervínculo, que se utiliza para enlazar a otras páginas web o recursos.

<a href="https://www.example.com">Visitar Ejemplo</a>

Etiqueta <img>

La etiqueta <img> se utiliza para insertar imágenes en la página web. Es una etiqueta auto-cerrada.

<img src="ruta/de/la/imagen.jpg" alt="Descripción de la imagen">

Etiqueta <footer>

La etiqueta <footer> se utiliza para mostrar información adicional o complementaria al final de la página. Puede incluir información de contacto, derechos de autor o enlaces a políticas.

<footer>
    <p>Derechos de autor © 2024. Todos los derechos reservados.</p>
</footer>

Etiqueta <section>

La etiqueta <section> se utiliza para agrupar elementos relacionados entre sí. Es útil para dividir la página en diferentes áreas o secciones, facilitando la organización del contenido.

<section>
    <h2>Sección de ejemplo</h2>
    <p>Este es un párrafo dentro de una sección.</p>
</section>

Etiqueta <article>

La etiqueta <article> se utiliza para definir contenido autónomo y coherente, como un blog, un artículo o una noticia. Es independiente del contenido del resto de la página y puede ser reutilizado o separado fácilmente.

<article>
    <h2>Título del artículo</h2>
    <p>Este es un artículo de ejemplo.</p>
</article>

Etiqueta <aside>

La etiqueta <aside> se utiliza para contenido complementario, como notas, enlaces relacionados o información adicional que no forma parte del contenido principal de la página.

<aside>
    <p>Notas o enlaces relacionados.</p>
</aside>

Etiqueta <blockquote>

La etiqueta <blockquote> se utiliza para citar el texto de otra fuente. A menudo se utiliza para citar frases o párrafos largos de otros textos o de la misma página.

<blockquote>
    <p>"La creatividad es la inteligencia divirtiéndose." - Albert Einstein</p>
</blockquote>

Etiqueta <code>

La etiqueta <code> se utiliza para incluir fragmentos de código fuente o código de programación. Es especialmente útil en páginas o secciones relacionadas con desarrollo y programación.

<code>
    <html></code>

Etiqueta <pre>

La etiqueta <pre> se utiliza para presentar texto en bloques, manteniendo la estructura y formato original del texto, como en código fuente o texto con sangrías o espacios.

<pre>
    <html></pre>
